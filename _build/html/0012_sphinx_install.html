Inštalácia #

Sphinx je multiplatformová aplikácia vytvorená v jazyku Python. Inštalácia pre rôzne platformy je popísaná v dokumentácii. Prehliadanie publikácií vytvorených pomocou Sphinx nevyžaduje okrem štandardného www prehliadača inštaláciu doplnkového programového vybavenia.

Linux #

Z Python Package Index (PyPI) pomocou inštalátora pip nainštalujeme základnú platformu Sphinx, podporu pre Markdown ako aj sadu základných užitočných rozšíreni:

pip install sphinx
pip install myst-parser
pip install myst-nb
pip install sphinx-copybutton
pip install sphinx-togglebutton
pip install sphinx-subfigure
pip install sphinx-design

  • myst-parser - podpora formátovania textov v Markdown na platforme Sphinx

  • myst-nb - rozšírenie myst-parser, umožňuje priame použitie Jupyter notebookov ako zdrojových textov

  • sphinx-copybutton - pridá do buniek textu formátovaných ako zdrojové kódy ikonu pre skopírovanie obsahu bunky do systémového zápisníka (clipboard)

  • sphinx-togglebutton - umožňuje skrývanie a rozbalovanie textu v poznámkach a upozorneniach

  • sphinx-subfigure - umožňuje ukladanie obrázkov do mriežky

  • sphinx-design - rozšírenie pre témy webových grafických rozhraní ako sú kaskádové štýly, expandovateľné kontainery, tlačítka, ikony a ďaľšie. Toto rozšírenie využívajú konfigurovateľné témy grafického rozhrania, ktoré sa inštalujú podľa potreby samostatne:

    • alabaster

    • sphinx-book-theme

    • pydata-sphinx-theme

    • sphinx-rtd-theme

    • furo

Témy sa inštalujú podľa potreby samostatne:

pip install alabaster
pip install sphinx-book-theme
pip install pydata-sphinx-theme
pip install sphinx-rtd-theme
pip install furo

V prípade potreby môžeme inštalovať paralelne niekoľko rôznych verzií Sphinx a ich konfigurácií vo virtuálnych prostrediach vytvorených pomocou venv. Aktuálnu verziu Sphinx získame pomocou príkazu

sphinx-build --version

Windows #

Python nie je štandardnou súčasťou distribúcie Windows, pre jeho inštaláciu na využite návody dostupné na internete, napríklad The Hitchhiker’s Guide to Python. Doporučená verzia pre inštaláciu je Python3. Po inštalácii nainštalujte Sphinx s príslušenstvom pomocou inštalátora pip rovnako ako pri inštalácii pre Linux.
